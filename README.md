# React Typescript Boilerplate

Includes:

-   Styled Components
-   Axios
-   Material Icons
-   ESLint
-   Prettier
-   Jest
-   PWA setup

Requires creating a `.env` file at the project root with a `REACT_APP_API_KEY` field which has an [OpenWeather API key](https://openweathermap.org/api) as its value.
